This project develops a fraud detection system using machine learning 
on the IEEE-CIS Financial Transaction Dataset. The objectives are to:

1. Perform exploratory data analysis on transactional and identity data
2. Engineer features to improve fraud signal detection
3. Handle class imbalance using SMOTE, ADASYN and RUS techniques
4. Train and evaluate multiple ML models (XGBoost, Random Forest, Logistic Regression)
5. Identify the best performing model based on AUC-ROC and F1-Score
6. Export results for Power BI dashboard visualization
